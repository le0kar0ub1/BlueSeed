# BlueSeed, the RISC-V emulator

*_BlueSeed is a RISC-V emulator for x64 plateform_*

## DEPENDENCIES

- *gcc*  for x86_64-elf target (dev under 9.2.1)
- *nasm*                       (dev under 2.14.02)
- *make*                       (dev under 4.3)

## INSTALL

_BlueSeed has been built whith a toolchain builder_

-   *make toolchain*

## BUILD

_make_


## RUN

_make run_
