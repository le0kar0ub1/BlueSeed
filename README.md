# BlueSeed, the RISC-V emulator

*_BlueSeed is a RISC-V emulator for x64 plateform__*

## DEPENDENCIES

- *gcc*  for x86_64-elf target (dev under 9.2.1)
- _nasm_                       (dev under 2.14.02)
- _make_                       (dev under 4.3)

## BUILD

_make_


## RUN

_make run_
